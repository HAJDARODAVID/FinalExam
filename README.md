## About the project

In this repo is the final exam - project for the BEDEV2023 course by Algebra. The task was to build a CMS application

## How to set up the project

- composer install
- npm install
- cp .env.example .env
- php artisan key:generate
- php artisan migrate
- php artisan db:seed
- npm run build

You will need:
- node -> 18.12.1
- npm -> 8.19.2
